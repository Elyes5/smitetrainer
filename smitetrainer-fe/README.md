# smitetrainer-fe

## Project setup

```
npm install
```

### Optional API base URL (defaults to http://localhost:8080)

```powershell
$env:VUE_APP_API_BASE_URL="http://localhost:8080"
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Main routes

- `/` Home
- `/multiplayer` Local multiplayer mode
- `/training` Backend-powered dragon HP analyzer
- `/about` Project info

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
